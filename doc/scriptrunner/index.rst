############
scriptrunner
############

`Commandable SAL Component`_ (CSC) to run Python scripts.

.. _Commandable SAL Component: https://docushare.lsst.org/docushare/dsweb/View/LSE-209

Scripts must be modules on the PYTHONPATH and the parent directory of the script must be named "scripts".
